¿Qué es Safe Docs?
Por: Safe Docs Team

Safe Docs es un a plataforma web basada en un software de reconocimiento facial, utilizado con el fin de garantizar la seguridad de los documentos e información que compartimos online. Esta iniciativa nace con la finalidad de propiciar la privacidad y controlar el acceso a los documentos compartidos entre los usuarios.

¿Cómo funciona?
Por: Safe Docs Team

Safe Docs nos permite acceder a documentos compartidos entre determinados usarios. Su funcionamiento se basa en el reconocimineto facial. El usuario deberá registrarse tanto para compartir documentos, como para acceder a los documentos correspondientes que se le hayan compartido. En el registro se le pedirá un nombre de usuario (username), nombre, apellidos y se tomará una ráfaga de fotos del usuario. Al ingresar a la plataforma, el usuario previamente registrado, deberá iniciar sesión. Una vez iniciada la sesión, para compartir documentos se deberá subir el documento correspondiente, y el username de la persona con la que quiera compartir. Los usuarios podrán visualizar todos los documentos que se le compartan una vez comprobada su identidad al iniciar sesión. Para iniciar sesión se hará uso de la cámara del usuario para poder evaluar mediante reconocimiento facil si la persona concuerda con con las fotos que se tomó en el registro previo.

¿Por qué usar Safe Docs?
Por: Safe Docs Team

Es una platforma amigable al usuario y sumamente facil de usar. Podras compartir y acceder a documentos de manera segura, priorizando la integridad y privacidad de la interacción online para compartir documentos. ¡Simplemente registrate y comienza a compartir información!
